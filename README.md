# Parallax Effect

Este projeto demonstra um efeito visual de parallax, utilizado para criar uma sensação de profundidade nas páginas da web. O efeito faz com que o conteúdo da página se mova a velocidades diferentes, criando uma experiência interativa e dinâmica para os usuários.

Embora o repositório esteja centrado no efeito visual de parallax, ele foi projetado com o propósito de ser facilmente adaptável a diferentes tipos de sites, incluindo sites de advocacia. O layout e o design podem ser facilmente personalizados para atender a outras necessidades, como sites institucionais, portais de serviços ou páginas de apresentação de escritórios de advocacia.

## Funcionalidades

- **Efeito Parallax**: O efeito é aplicado de maneira simples e eficaz nas imagens de fundo da página.
- **Responsividade**: O layout é adaptável a diferentes tamanhos de tela, proporcionando uma boa experiência tanto em desktops quanto em dispositivos móveis.
- **Estrutura de Site**: O repositório contém uma página inicial e várias páginas adicionais que podem ser usadas para expandir o conteúdo do site.

## Tecnologias Utilizadas

- **HTML**: Para a estruturação da página.
- **CSS**: Para o design visual e aplicação do efeito parallax.
- **JavaScript**: Para o controle do movimento das imagens e criação da interação.

## Como Usar

1. Clone este repositório para a sua máquina local:
    ```bash
    git clone https://github.com/ogabrieldias/paralax-effect.git
    ```
2. Abra o arquivo `index.html` no seu navegador para visualizar o efeito parallax.
3. Personalize o conteúdo conforme necessário, especialmente a parte do design, que pode ser facilmente adaptada para um site de advocacia ou outro tipo de negócio.
